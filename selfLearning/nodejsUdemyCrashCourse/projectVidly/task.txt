54. Managing the list of "genre"s.
	url: http://vidly.com/api/genres
	Each movie has a "genre" like action, horror, etc.
	You should have an endpoint for following
		You should be able to get list of all "genre".
		You should be able to create a new "genre".
		You should be able to update a "genre".
		You should be able to delete a "genre".
